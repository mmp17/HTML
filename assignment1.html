  • !DOCTYPE
    • Ushbu deklaratsiya hujjat turini ifodalaydi va brauzerlarga veb sahifalarni to'g'ri ko'rsatishga yordam beradi.
  • HTML element
    • Bu element asosiy element bo'lib, u HTML to'liq hujjatni belgilaydi.
  • Body element
    • Ushbu BODY elementi hujjatning asosiy qismi, ya'ni uning tanasini belgilaydi.
  • HTML Headings
    • HTML sarlavhalari h1 dan h6 gacha taglari bilan belgilanadi. h1 eng muhim sarlavhani, h6 esa eng muhum bo'lmagan sarlavhani belgilaydi.
  • HTML Paragraphs
    • HTML paragraflari p tegi bilan belgilanadi
  • HTML Links
    • HTML havolalari a tegi bilan belgilanadi. Teg giperhavolani belgilaydi. Href atributi havola qilinadigan sahifaning URL manzilini ko'rsatadi
  • HTML Images
    • HTML rasmlari img tegi bilan belgilanadi. Manba fayli (src), muqobil matn (alt), kenglik va balandlik atributlar sifatida taqdim etiladi. Img tegi tasvirni HTML sahifasiga joylashtirish uchun ishlatiladi. Src xususiyati ko'rsatiladigan rasmga yo'lni belgilaydi. Img tegi kenglik va balandlik atributlarini ham o'z ichiga olishi kerak. Bu rasmning kengligi va balandligini piksellarda belgilaydi. Img tegi uchun zarur bo'lgan alt atributi esa agar rasm biron sababga ko'ra ko'rsatilmasa, rasm uchun muqobil matndir. Buning sababi sekin internet tezligi bo'lishi mumkin yoki src atributidagi xatolik bo'lsa yoki agar foydalanuvchi kurishi cheklangan bo'lib ekran o'quvchi (screen reader) ishlatsa.
  • HTML Quotation and Citation Elements
    • HTML blockquote elementi boshqa manbadan keltirilgan bo'limni belgilaydi. HTML q tegi boshqa manbadan olingan malumotni belgilaydi. HTML abbr tegi "HTML", "CSS" "ATM" kabi qisqartmalarni aniqlaydi. Qisqartirishlarni belgilash brauzerlar, tarjima tizimlari va qidiruv tizimlariga foydali ma'lumotlarni berishi mumkin. HTML manzil tegi hujjat yoki maqola muallifi/egasining aloqa ma'lumotlarini belgilaydi.
